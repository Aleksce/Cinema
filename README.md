# Cinema Booking (WPF + SQLite)

Готовый учебный шаблон приложения для онлайн-бронирования билетов в кинотеатр на **WPF** с использованием **SQLite** через **Entity Framework Core**.

## Что реализовано

- Регистрация и вход пользователя.
- Хранение пользователей в БД.
- Список сеансов.
- Загрузка мест по выбранному сеансу.
- Бронирование свободного места.
- Просмотр собственных бронирований.
- Сохранение данных между запусками (`cinema.db`).

## Стек

- .NET 8
- WPF
- Entity Framework Core
- SQLite

## Структура проекта

- `CinemaBooking.App/Models` — модели (`User`, `MovieSession`, `Seat`, `Booking`).
- `CinemaBooking.App/Data` — `CinemaDbContext` и инициализация БД.
- `CinemaBooking.App/Services` — сервисы авторизации и бронирования.
- `CinemaBooking.App/ViewModels` — MVVM-логика главного окна.
- `CinemaBooking.App/MainWindow.xaml` — UI приложения.

## Быстрый старт

1. Откройте проект в Visual Studio 2022+ на Windows.
2. Восстановите NuGet-пакеты.
3. Запустите приложение.
4. При первом запуске автоматически создастся файл `cinema.db` и тестовые сеансы.
